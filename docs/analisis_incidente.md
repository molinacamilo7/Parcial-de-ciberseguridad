# Análisis del incidente de seguridad

En el escenario planteado se identifican varias señales que indican la presencia de un ataque informático.

En primer lugar, algunos archivos importantes aparecen cifrados con una extensión desconocida. Este comportamiento suele estar relacionado con ataques de ransomware.

Además, los equipos muestran un mensaje solicitando un pago en criptomonedas para recuperar la información.

También se observan conexiones hacia direcciones IP sospechosas en los registros del sistema, lo cual puede indicar comunicación con servidores externos controlados por los atacantes.

Finalmente se identifica que el incidente comenzó cuando un empleado descargó un archivo adjunto que aparentemente provenía de un proveedor, pero que en realidad contenía un archivo malicioso.

Este tipo de situaciones suele ocurrir cuando los atacantes utilizan técnicas de phishing para engañar a los usuarios.
